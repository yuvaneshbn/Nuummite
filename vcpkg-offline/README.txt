This folder is for offline vcpkg installation support.

To use vcpkg offline:
1. Download the vcpkg source ZIP from https://github.com/microsoft/vcpkg and extract it here.
2. Download required ports/packages and place them in the 'downloads' folder inside vcpkg.
3. Follow the instructions in the main README for offline installation.

Do NOT commit vcpkg binaries or large files to GitHub. For full offline support, create a ZIP archive and upload it as a GitHub release attachment.

For more details, see https://github.com/microsoft/vcpkg/blob/master/docs/users/offline.md
